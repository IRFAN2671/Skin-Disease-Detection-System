# Skin Disease Detection & Triage System

Final Year Project (FYP) - University of Management and Technology (UMT)

## 📌 Project Overview
An intelligent deep learning pipeline designed to classify skin lesions and perform medical triage profiling. The system leverages state-of-the-art Convolutional Neural Networks (CNNs) to analyze dermatological images and categorize them based on clinical urgency, providing seamless interfaces for both patients and healthcare experts.
